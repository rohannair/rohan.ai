# Welcome

This is my personal website. It's built using [SvelteKit](https://kit.svelte.dev/) (Svelte 5), [Tailwind CSS](https://tailwindcss.com/) (v4) and ingenuity.

## Tech Stack

*   [SvelteKit](https://kit.svelte.dev/) & [Svelte 5](https://svelte.dev/)
*   [Tailwind CSS v4](https://tailwindcss.com/)
*   [UnoCSS](https://unocss.dev/) for a faster development runtime
*   [Shiki](https://shiki.style/) for Markdown syntax highlighting
*   [Vite 7](https://vitejs.dev/)

## Development

This project uses `pnpm` for package management and `bun` for scripts.

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Check types and code
pnpm check

# Create a new blog post
bun run scripts/newPost.ts
```
